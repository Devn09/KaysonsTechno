# Kaysons Techno Equipments website

Static four-page website for GitHub Pages. No npm, Node.js or Render is required.

Pages: Home, Product Portfolio, About Us and Contact.

The package includes a branded first-visit preloader, scroll-progress indicator, section reveals, staggered card animation and reduced-motion accessibility. The loader is skipped automatically when navigating between pages in the same browser session. Open index.html locally to review the site. The enquiry form is a visual demo until a form service is connected.
