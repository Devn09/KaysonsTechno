# Kaysons Techno Equipments Private Limited website — version 2

Static four-page GitHub Pages website.

## Pages

- `index.html` — Homepage
- `products.html` — Fourteen explosionproof product families
- `about.html` — Company overview
- `contact.html` — Contact details and visual enquiry form

## Shared files

- `styles.css` — Core industrial design system
- `site.css` — Page layouts, typography and responsive rules
- `script.js` — Mobile menu, current year and demo-form message
- `kaysons-logo.jpg` — Supplied company logo

No build command, Node.js, npm or Render service is required. Publish the files from the root of a GitHub repository using GitHub Pages.

The enquiry form is visual only and does not send data. Connect it to a chosen form service or backend before enabling live submissions. The certification and approval marks are prominent typographic placeholders; replace them with approved official artwork when supplied and verify scope against the applicable product/model.

See `GITHUB_DESKTOP_WINDOWS.md` for publishing instructions.
