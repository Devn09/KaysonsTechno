# Kaysons three-page website

Static GitHub Pages website for Kaysons Techno Equipments Pvt. Ltd.

## Pages

- `index.html` — Homepage
- `products.html` — Product portfolio with temporary content
- `contact.html` — Contact / requirement page with temporary information and a demo form

## Shared files

- `styles.css` — Core industrial design system
- `site.css` — Three-page layout and responsive extensions
- `script.js` — Mobile menu, current year and demo-form message
- `kaysons-logo.jpg` — Supplied Kaysons logo

No build command, Node.js, npm, Render or database is required. Open `index.html` locally or publish the files from the root of a GitHub repository using GitHub Pages.

Important: the enquiry form is visual only. Product details, official contact information, approval artwork and verified certification scope must be completed before public launch.

See `GITHUB_DESKTOP_WINDOWS.md` for publishing instructions.
